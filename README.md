# 中国移动资费公示桌面爬取工具

> 基于 Python + PySide6 + Selenium 的桌面 GUI 应用，自动抓取中国移动资费公示页面的套餐数据，支持导出 Excel。

## 功能亮点

- **可视化浏览器** — Chrome 浏览器可见，用户手动选择省份
- **全自动遍历** — 自动切换所有资费类型（套餐、加装包、营销活动），逐个抓取
- **智能滚动** — 自动滚到底部触发懒加载，精确判断是否加载完毕
- **实时预览** — 数据实时显示在表格中，支持关键字搜索和价格筛选
- **一键导出** — 导出带格式的 Excel，18 列完整字段
- **打包部署** — 支持打包成 EXE，其他电脑无需安装 Python

## 技术栈

| 技术 | 用途 |
|------|------|
| Python 3.8+ | 开发语言 |
| PySide6 | GUI 桌面界面 |
| Selenium | 浏览器自动化控制 |
| openpyxl | Excel 文件导出 |
| PyInstaller | 打包成可执行文件 |

## 快速开始

### 环境要求

- Python 3.8 或更高版本
- Google Chrome 浏览器（最新版）

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python cmcc_fee_crawler.py
```

### 打包成 EXE

```bash
# 方式一：双击 build.bat
build.bat

# 方式二：手动打包
pip install pyinstaller
pyinstaller --noconfirm --onedir --windowed --name "中国移动资费公示爬取工具" cmcc_fee_crawler.py
```
