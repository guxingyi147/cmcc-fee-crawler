# 中国移动资费公示桌面爬取工具 v2.0

> 基于 Python + PySide6 + Selenium 的桌面 GUI 应用，自动抓取中国移动资费公示页面的套餐数据，支持导出 Excel。

## 功能亮点

- **可视化浏览器** — Chrome 浏览器可见，用户手动选择省份
- **全自动遍历** — 自动切换所有资费类型（套餐、加装包、营销活动），逐个抓取
- **智能滚动** — 自动滚到底部触发懒加载，精确判断是否加载完毕
- **实时预览** — 数据实时显示在表格中，支持关键字搜索和价格筛选
- **一键导出** — 导出带格式的 Excel，18 列完整字段
- **日志放大** — 支持全屏查看日志，方便监控抓取过程
- **自动驱动管理** — Selenium Manager 自动匹配 ChromeDriver，无需手动下载
- **打包部署** — 支持打包成 EXE，其他电脑无需安装 Python

## 技术栈

| 技术 | 用途 |
|------|------|

## 快速开始

### 环境要求

- Python 3.8 或更高版本
- Google Chrome 浏览器（最新版）

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python main.py
```

### 打包成 EXE

```bash
# 方式一：双击 build.bat
build.bat

# 方式二：手动打包
pip install pyinstaller
python -m PyInstaller --noconfirm --onedir --windowed --name "cmcc_crawler" main.py
```

## 数据字段

共 18 列：方案编号、套餐名称、资费标准、资费类型、适用地区、适用范围、销售渠道、上线日期、下线日期、有效期限、在网要求、**国内通话**、**国内通用流量**、**定向流量**、**短信**、退订方式、违约责任、其他服务内容

## ChromeDriver 自动管理

程序使用 Selenium Manager 自动管理 ChromeDriver，无需手动下载：

1. 程序启动时自动检测本地 Chrome 版本
2. 从国内镜像（npmmirror）自动下载匹配的 ChromeDriver
3. 缓存到 `~/.cache/selenium`，下次运行直接复用
4. Chrome 更新后自动下载新版本驱动
