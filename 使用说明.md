# 使用说明

## 一、运行前准备

### 1.1 安装 Python

1. 前往 https://www.python.org/downloads/ 下载 Python 3.8+
2. 运行安装程序，**务必勾选 "Add Python to PATH"**
3. 安装完成后，打开命令提示符验证：
   ```
   python --version
   ```
   显示版本号即安装成功。

### 1.2 安装 Chrome 浏览器

1. 前往 https://www.google.com/chrome/ 下载安装
2. 确保为最新版本（程序会自动匹配 ChromeDriver）

### 1.3 安装依赖库

打开命令提示符，进入项目目录，执行：
```
pip install PySide6 selenium openpyxl
```

或直接：
```
pip install -r requirements.txt
```

### 1.4 ChromeDriver 说明

程序会自动处理 ChromeDriver，通常无需手动操作。

**自动获取顺序：**
1. 检查程序同目录下是否有 `chromedriver.exe`
2. 检查本地缓存 `C:\Users\用户名\.cache\cmcc_crawler\`
3. 从国内镜像自动下载匹配版本
4. 回退到 Selenium 内置驱动管理

**手动下载（仅在自动失败时）：**
1. 打开 Chrome → 设置 → 关于 Chrome，记录版本号
2. 访问 https://googlechromelabs.github.io/chrome-for-testing/
3. 找到匹配版本的 chromedriver，下载 `win64/chromedriver-win64.zip`
4. 解压得到 `chromedriver.exe`，放到 `cmcc_fee_crawler.py` 同目录

## 二、操作步骤

### 2.1 启动程序

```
python cmcc_fee_crawler.py
```

程序启动后会显示桌面窗口。

### 2.2 启动浏览器

点击 **「启动浏览器」** 按钮。

- 程序自动启动 Chrome，打开中国移动资费公示页面
- 日志区会显示启动进度
- 如果是首次运行，可能需要几秒下载 ChromeDriver

### 2.3 选择省份

**在弹出的 Chrome 浏览器中**，手动选择你需要的省份。

- 页面顶部有省份选择区域
- 点击目标省份即可切换
- 省份选择后页面会自动刷新显示该省的套餐

> 注意：程序不会自动操作省份选择，需要用户手动完成。

### 2.4 开始抓取

回到软件界面，点击 **「开始抓取」** 按钮。

程序会自动执行：
1. 读取资费类型下拉框的所有选项（如套餐、加装包、营销活动）
2. 逐个切换资费类型
3. 每个类型下自动滚动页面到底部，触发懒加载
4. 解析页面上的所有套餐卡片
5. 以「方案编号」自动去重
6. 实时将数据显示在表格中

**抓取过程中你可以：**
- 在浏览器中观察页面自动滚动
- 在软件表格中实时查看已抓取的数据
- 在日志区查看详细进度
- 随时点击「停止抓取」中断（已抓数据不丢失）

### 2.5 搜索和筛选

抓取过程中或抓取完成后，可以随时使用搜索筛选功能：

**关键字搜索：**
- 在搜索框输入关键词（如"全球通"或"25JT"）
- 支持按套餐名称和方案编号模糊匹配
- 输入即时生效，无需点击按钮

**价格区间筛选：**
- 在最低价和最高价输入框填写数字
- 点击「筛选」按钮生效
- 点击「重置」清除所有筛选条件

### 2.6 导出 Excel

1. 点击 **「导出Excel」** 按钮
2. 选择保存位置和文件名
3. 等待导出完成

**Excel 文件特性：**
- 18 列完整字段，表头冻结
- 自动筛选功能（可按列排序和筛选）
- 交替行颜色，便于阅读
- 4 个新增字段（国内通话、国内通用流量、定向流量、短信）用特殊颜色高亮
- 列宽已优化，自动换行

### 2.7 清空数据

点击 **「清空数据」** 按钮，会弹出确认对话框，确认后清除所有已抓取的数据。

> 注意：清空后数据无法恢复，建议先导出 Excel 再清空。

## 三、打包成 EXE

### 3.1 打包步骤

```bash
# 方式一：双击 build.bat（推荐）
build.bat

# 方式二：手动执行
pip install pyinstaller
pyinstaller --noconfirm --onedir --windowed --name "中国移动资费公示爬取工具" cmcc_fee_crawler.py
```

打包完成后，可执行文件位于：
```
dist\中国移动资费公示爬取工具\中国移动资费公示爬取工具.exe
```

### 3.2 在其他电脑上使用

**前置要求：仅需安装 Google Chrome 浏览器。**

不需要安装 Python、不需要安装任何依赖库。

**部署步骤：**
1. 将整个 `dist\中国移动资费公示爬取工具\` 文件夹复制到目标电脑
2. 双击 `中国移动资费公示爬取工具.exe` 运行
3. 首次运行会自动下载 ChromeDriver（需联网，约几秒）

> 必须保持整个文件夹完整，不能只复制 .exe 单独运行。

**离线部署（无网络）：**
1. 在有网络的电脑上运行一次程序
2. 复制 `C:\Users\用户名\.cache\cmcc_crawler\` 文件夹到目标电脑相同路径
3. 或将 `chromedriver.exe` 直接放到 `.exe` 同目录下

## 四、常见问题

### Q: 提示"python 不是内部命令"
A: Python 未安装或未添加到 PATH。重新安装 Python，勾选 "Add Python to PATH"。

### Q: 启动浏览器失败
A: 确保 Chrome 已安装且为最新版。检查是否有杀毒软件拦截。

### Q: ChromeDriver 下载失败
A: 手动下载 chromedriver.exe 放到程序同目录（详见 1.4 节）。

### Q: 抓取数据为空
A: 确保页面完全加载后再点击「开始抓取」。检查是否选择了有数据的省份。

### Q: 表格数据不完整
A: 可能页面还没加载完就判定到底。重新抓取即可，程序会自动去重。

### Q: 导出 Excel 为空
A: 需要先抓取到数据才能导出。

### Q: 打包后双击没反应
A: 首次启动可能较慢（解压文件），等待几秒。检查杀毒软件是否拦截。

### Q: 打包后找不到 chromedriver
A: 将 chromedriver.exe 放到 .exe 同目录下。
