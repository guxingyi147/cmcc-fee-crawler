# 项目说明

## 一、开发背景

中国移动资费公示页面（https://www.10086.cn/fee/）公开了各省的套餐、加装包、营销活动等资费信息。但由于：

1. **数据分散** — 不同资费类型（套餐、加装包、营销活动）分在不同分类下
2. **懒加载** — 页面采用滚动加载，每次只显示少量数据
3. **无法批量导出** — 官网不提供数据导出功能
4. **人工整理效率低** — 需要逐页浏览、手动记录

因此开发本工具，实现自动化批量抓取，一键导出结构化 Excel。

## 二、核心功能

### 2.1 功能清单

| 功能 | 说明 |
|------|------|
| 启动浏览器 | 自动启动 Chrome 并打开资费公示页面 |
| 手动选省份 | 用户在浏览器中手动选择目标省份 |
| 自动遍历资费类型 | 自动切换套餐、加装包、营销活动等所有类型 |
| 自动滚动抓取 | 滚动到底部触发懒加载，逐批抓取数据 |
| 实时数据展示 | 抓取的数据实时显示在表格中 |
| 关键字搜索 | 按套餐名称或方案编号模糊搜索 |
| 价格区间筛选 | 按资费标准筛选指定价格范围 |
| 自动去重 | 以方案编号为唯一 ID，重复数据不存入 |
| 导出 Excel | 导出带格式的 .xlsx 文件，18 列完整字段 |
| 打包部署 | 支持打包成 EXE，在其他电脑直接运行 |

## 三、技术原理

### 3.1 整体架构

```
┌─────────────────────────────────────────────┐
│              MainWindow (主线程)               │
│  按钮 / 搜索 / 表格 / 状态栏 / 日志           │
└──────────────────┬──────────────────────────┘
                   │ PySide6 信号
┌──────────────────▼──────────────────────────┐
│           CrawlerThread (子线程)              │
│  遍历资费类型 → 滚动页面 → JS提取 → 解析去重  │
└──────────────────┬──────────────────────────┘
                   │ Selenium WebDriver
┌──────────────────▼──────────────────────────┐
│              Chrome 浏览器 (可视化)            │
└─────────────────────────────────────────────┘
```

### 3.2 线程模型

- **主线程**：只负责 GUI 显示和用户交互，不做任何 Selenium 操作
- **爬虫线程**：负责所有浏览器交互和 DOM 数据提取

两个线程通过 PySide6 信号通信：
- `new_data(list)` — 新数据发送到主线程显示
- `log(str)` — 日志信息
- `status(str)` — 状态更新
- `type_switched(str)` — 资费类型切换通知
- `finished()` — 抓取完成

### 3.3 页面 DOM 结构

#### 资费类型下拉框

```
div.line-3
  ├── div.select-label "资费类型："
  ├── div.select-container.the-select  ← 第0个 = 资费类型
  │   ├── div.select-box
  │   │   ├── div.tipsText "套餐"     ← 当前选中值
  │   │   └── div.select-icon
  │   └── div.select-list-box          ← 展开后的选项
  │       ├── div.select-item[data-value="1"]  → 套餐
  │       ├── div.select-item[data-value="2"]  → 加装包
  │       ├── div.select-item[data-value="3"]  → 营销活动
  │       └── div.select-item[data-value="4"]  → 国际/港澳台资费
  ├── div.select-label "资费系列："
  └── div.select-container.the-select  ← 第1个 = 资费系列
```

#### 套餐卡片

```
div.tariff-item-container  ← 精确选择器，每个套餐一个
  ├── div.item-name "全球通5GA尊享套餐299档"  ← 套餐名称
  └── div.item-tips-list
      ├── 资费标准: 299元/月
      ├── 方案编号: 25JT205474
      ├── 资费类型: 套餐
      ├── 适用范围: 全网用户
      ├── ...
      ├── 国内通话: 1000分钟
      ├── 国内通用流量: 140GB
      └── ...
```

### 3.4 数据提取流程

```
1. JS 批量提取
   └── execute_script("querySelectorAll('.tariff-item-container')")
       → 返回所有卡片的 innerText 和 .item-name 文本

2. 纯 Python 解析
   └── 正则匹配 "label：value" 格式
       → 提取 18 个字段

3. 去重
   └── 以方案编号为 key，已存在则跳过

4. 信号发送
   └── new_data.emit(batch) → 主线程追加到表格
```

### 3.5 滚动与加载检测

```
每轮循环：
  1. scrollTo(0, document.body.scrollHeight)  → 滚到底部
  2. sleep(1秒)                                → 等待加载
  3. JS querySelectorAll 获取卡片数量           → 毫秒级
  4. 数量变化？→ 是：解析新数据，继续
              → 否(连续3次)：该类型抓取完毕
```

### 3.6 ChromeDriver 获取策略

```
优先级 1: 程序同目录/chromedriver.exe
优先级 2: ~/.cache/cmcc_crawler/chromedriver_版本号/chromedriver.exe
优先级 3: 通过 Google API 查询匹配版本 → npmmirror 国内镜像下载
优先级 4: Selenium 内置驱动管理（需翻墙）
```
